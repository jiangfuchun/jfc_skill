# 上市公司深度研报引擎

## 核心原则

本 skill 整合 **8 套经过长期验证的分析方法论**，不自创框架。每套方法论各有适用场景，不为了凑数全用，**按公司的行业阶段 + 业务特征选出最适用的 4-6 套组合使用**。

## 工作流

### Step 0：时间锚定（必须执行）

在开始任何操作之前，使用 `date` 命令或系统提示信息获取**当前准确日期和时间**。当前日期决定了搜索的时间范围（如搜索"最近 3 年财务数据"需以当前日期为基准推算年份）以及报告的输出日期。

### Step 1：信息采集（必须执行）

在开始任何分析之前，先用搜索工具收集以下信息。所有搜索查询中的年份参数需根据 Step 0 获取的当前日期动态生成（例如今天是 2026-06-25，则"最近 3 年"指 2023-2025 完整财年 + 2026 上半年可用数据，"2026 年"指当前年份）：

```
1. 财务数据：最近 3 年营收、净利润、毛利率、净利率、ROE、经营性现金流
2. 股价与估值：最新股价、PE(TTM)、PB、EV/EBITDA、总市值、52周高低
3. 业务结构：各业务板块收入拆分、核心客户、前五大客户占比
4. 行业数据：所在行业的市场规模、增速、渗透率、竞争格局
5. 管理层：CEO/董事长背景、持股比例、管理层变动、股权激励
6. 重大事项：近期公告、再融资、并购、诉讼、监管事件
7. 券商覆盖：最近 3 个月的目标价区间、评级分布
8. ESG 信息：重大环境/安全事件、碳排放、治理结构、分红历史
9. 股东结构：前十大股东、机构持仓变动、港股通/北上资金流向
```

信息不足的直接标注"待补充"，不编造。关键财务数据必须在正文中注明数据来源。

### Step 1：行业生命周期定位

用行业生命周期理论判断公司所在赛道的阶段，这决定了后续使用哪些分析工具和估值方法。

**分析框架：**

| 阶段 | 特征 | 适用估值方法 | 关键指标 |
|------|------|-------------|---------|
| 导入期 | 渗透率 <5%，无盈利，高研发 | PS / 用户估值 | 用户增速、单位经济模型(UE) |
| 成长期 | 渗透率 5-30%，营收高速增长，开始盈利 | PEG / PS / DCF | 营收增速、毛利率趋势 |
| 成熟期 | 渗透率 >30%，增长放缓，现金牛 | PE / DCF / EV/EBITDA | ROE、自由现金流、分红率 |
| 衰退期 | 负增长，份额萎缩 | PB / 清算估值 | 资产价值、收缩计划可信度 |

**渗透率 S 曲线判断：**
- 如果赛道是技术替代型（新能源车替代燃油车/液冷替代风冷）：标注当前渗透率位置，找到**拐点区间**
- 如果赛道是创造型（AI 原生应用）：标注是 0→1 还是 1→100

> **输出要求：** 写明行业当前渗透率（有数据则引用），判断处于 S 曲线的哪个位置，并说明判断依据。**所有时间相关的判断（"未来 3 年"、"当前增速"等）必须以 Step 0 获取的当前时间为基准。**

### Step 2：波特五力分析

#### 适用场景
行业结构化、竞争格局稳定的赛道（消费/制造/金融等）。纯 0→1 的超级新兴赛道（无竞争结构时）可以简用或跳过。

#### 五力模型

**1. 现有企业间的竞争强度**
- 行业集中度（CR3/CR5/HHI 指数）——有数据必须引用
- 产品差异化程度（同质化 → 价格战，差异化 → 利润保护）
- 退出壁垒（产能退出成本高 → 容易恶性竞争）
- 行业增速（增速快 → 暂时掩盖竞争，增速慢 → 零和博弈）

**2. 潜在进入者威胁**
- 规模经济门槛、资本门槛、技术门槛
- 政策壁垒（牌照、资质）
- 现有企业的报复能力（先发者优势、转换成本）
- 最近 2 年是否有重要新进入者

**3. 替代品威胁**
- 替代品性价比趋势（比现有方案更好/更便宜？）
- 用户的转换意愿和成本
- 技术路线是否收敛（多路线并行 → 替代风险高）

**4. 供应商议价能力**
- 核心原材料/零部件供应商集中度
- 公司对供应商的依赖程度（单一来源 vs 多源供应）
- 前五大供应商占比

**5. 买方议价能力**
- 客户集中度（前五大客户占比 >30% → 买方议价能力强）
- 产品对客户的重要性（必需品 vs 可选品）
- 客户转换成本

> **输出要求：** 每个维度给出明确的高/中/低判断，最后给出**五力综合评分**（竞争格局好 / 中等 / 恶劣）。标明凭什么数据做出判断。

### Step 3：护城河分析（晨星框架）

源自 Pat Dorsey《巴菲特的护城河》，识别公司的结构性竞争优势。

**五种护城河来源（逐一评估）：**

**1. 无形资产（品牌/专利/牌照）**
- 品牌：品牌是否影响用户购买决策（高品牌溢价的才是真品牌）
- 专利：核心专利的剩余保护期和被无效的风险
- 牌照/特许权：监管壁垒（如金融牌照、资源开采权），是否排他

**2. 转换成本**
- 用户的替代成本是否足够高（包括：数据迁移成本、培训成本、业务流程重构成本）
- 典型案例：企业软件（SAP/Oracle）、银行的核心系统

**3. 网络效应**
- 需求的增加是否提升了每个用户的价值
- 单边 vs 多边平台（双边市场网络效应更强）
- 关键判断点：网络效应是否"自然的"而非"烧钱维持的"

**4. 成本优势**
- 结构性的低成本（流程创新、地理位置、规模、独特资源），而非靠压缩毛利的价格战
- 低成本能否维持（竞争对手模仿的难度和时间）

**5. 规模经济（有效规模）**
- 市场容量有限，一家或几家大企业已经用低成本填满了需求
- 新进入者即使建厂也达不到规模效应所需的成本和价格

> **输出要求：** 列出公司实际拥有的护城河，每一条写明是"宽/窄/无"。对声称有护城河的公司，追问：这个优势能不能撑 10 年？竞争对手花多少钱、多少时间能复制？

### Step 4：财务健康度——杜邦分析 + 现金流质量

#### 杜邦分解 ROE

```
ROE = 净利率 × 资产周转率 × 权益乘数（财务杠杆）
```

分析逻辑：
- **高净利率型**（茅台海天等）：毛利率是核心，关注护城河能否支撑高毛利
- **高周转型**（沃尔玛/制造企业）：运营效率是核心，关注存货/应收周转天数
- **高杠杆型**（银行/地产）：风险是核心，关注杠杆倍数和资产质量

**实操步骤：**
1. 计算最近 3 年的 ROE 及三级分解
2. 判断公司的盈利模式属于哪一型
3. 与行业均值对比，找出"异常"（显著高于/低于同行的指标）
4. 分析异常的可持续性

#### 现金流质量检查

- 经营性现金流是否长期 > 净利润？（>1.0 健康，<0.7 警惕）
- 自由现金流（经营现金流 - 资本开支）是否持续为正或转正趋势？
- 资本开支与折旧之比（>2 说明仍在高速扩张期，<1 说明在"吃老本"）
- 应收账款周转天数是否逐年上升？（↑ 可能是压货或放松信用政策）
- 存货周转天数趋势（↑ 可能产品竞争力下降）

#### ⚠️ 财务警示信号检查清单

逐项检查并在发现异常时特别标注：

| 警示信号 | 表现 | 严重性 |
|---------|------|:------:|
| 营收增长但经营现金流连续下降 | "纸面利润" | ⚪⚪⚪ |
| 应收账款增速远超营收增速 | 压货/回款困难 | ⚪⚪⚪ |
| 存货周转天数大幅上升 | 产品滞销风险 | ⚪⚪⚪ |
| 商誉占比 >30% 净资产 | 并购减值风险 | ⚪⚪⚪ |
| 大股东大比例质押 (>50%) | 控制权/资金占用风险 | ⚪⚪⚪ |
| 频繁更换审计机构 | 财务真实性存疑 | ⚪⚪⚪ |
| 关联交易占比异常高 | 利润操纵空间 | ⚪⚪⚪ |
| 研发投入资本化率异常高 | 虚增利润 | ⚪⚪⚪ |
| 长期应付款远大于账面利润 | 隐性债务/表外负债 | ⚪⚪⚪ |
| 高管集中套现 | 内部人信号 | ⚪⚪⚪ |

> **输出要求：** 至少完成近 3 年 ROE 杜邦分解、经营现金流/净利润比值、自由现金流趋势、并检查警示信号清单。发现任何警示信号须特别高亮。给出财务健康评级（健康 / 关注 / 高风险）。

### Step 5：管理层评估框架

一套实用的管理层评估维度，不涉及心理分析，只做可验证的观察：

**1. 历史言行一致性**
- 3 年以上的目标承诺与实际结果对比：说到做到的比例
- 并购/收购后的效果：收购时画的饼实现了多少
- 股权激励的行权条件是否合理，还是"送钱"

**2. 资本配置能力**
- 主业增长和资金消耗的效率（投入资本回报率 ROIC vs 加权资金成本 WACC）
- 再融资记录：是"缺钱融资"还是"扩张融资"
- 分红和回购历史：是否善待股东

**3. 治理结构**
- 实际控制人持股比例（太高→一言堂风险，太低→代理人风险）
- 独立董事占比和履职意愿（是否有真正不同意见的记录）
- 关联交易：定价公允性、频率、金额

**4. 专业化 vs 多元化**
- 主业做透了还是分心了？（营收增长的同时非主业投资是否增加）
- 多元化是"能力延伸"还是"故事拼凑"？

> **输出要求：** 基于可查事实给出管理层评分（强 / 一般 / 存疑），每项判断都引证据。避免"优秀的管理层"这类空话。

### Step 6：估值矩阵

#### 方法选择指南

根据 Step 1 的行业阶段，选择 2-3 种最适用的估值方法：

| 方法 | 适用场景 | 核心输入 | 关键陷阱 |
|------|---------|---------|---------|
| **DCF** | 成熟/稳定现金流公司（消费/公用/传统制造） | FCF 预测、WACC、永续增速 | 终值占比过高 → 微小假设变化=巨大估值差异 |
| **PE** | 盈利稳定的成长/成熟公司 | 可比公司 PE、合理 PE 中枢 | 周期股在高点 PE最低时最贵 |
| **PEG** | 盈利增速 >15% 的成长股 | PE / 净利润增速 | 增速回落时双杀 |
| **PS** | 早期/高增长但未盈利的科技公司 | 可比 PS、稳态利润率假设 | PS 完全不考虑利润质量 |
| **PB** | 金融/周期/重资产公司 | 净资产、ROE | 无形资产多的公司 PB 失效 |
| **EV/EBITDA** | 资本密集型企业/杠杆差异大的公司 | EV、EBITDA、可比倍数 | EBITDA 忽略资本开支和税务差异 |
| **SOTP** | 多元化集团 | 各业务分部估值求和 | 内部交易和折价系数的主观性 |

#### 情景分析结构

对选择的估值方法，做三情景预测：

| 情景 | 假设条件 | 触发条件 |
|------|---------|---------|
| **保守** | 行业增速放缓，公司份额保持稳定 | 宏观下行/行业竞争加剧 |
| **中性** | 行业维持当前增速，公司份额略升 | 当前趋势延续 |
| **乐观** | 行业超预期 + 公司份额明显提升 | 新产品/新市场/政策催化 |

> **输出要求：** 给出**保守/中性/乐观三情景目标价**，并注明各情景需要什么条件才能达到。所有估值模型输入参数必须写明假设依据（"假设 WACC=10%"是不够的，要说明为什么是 10%）。

### Step 7：ESG 评估（可选/成熟期和金融业必须）

**E - 环境：**
- 是否属于高耗能行业？碳排放强度和同行对比
- 环保处罚记录（重点关注化工/能源/制造行业）
- 碳中和路线图的清晰度和可信度

**S - 社会：**
- 重大安全事故记录
- 员工关系（劳动纠纷/社保合规）
- 产品责任（客户投诉/质量召回）

**G - 治理：**（Step 5 已覆盖治理部分，这里做补充）
- 分红政策连贯性
- 信息披露质量（交易所考核结果）
- 诉讼纠纷（小股东诉讼、监管处罚记录）

> **输出要求：** 列出重大 ESG 风险点，判断是否构成投资障碍。ESG 不应被用作"加分项"打分，而是作为**排除项**——有重大 ESG 问题的公司直接标注风险等级。

### Step 8：SWOT 总成

整合以上所有分析，用一个统一的 SWOT 矩阵做收束。注意：SWOT 是总结，不是分析起点。

**要求：**
- 优势和劣势都是**内部因素**（公司能控制的）
- 机会和威胁都是**外部因素**（公司不能控制的）
- 每个条目必须是可验证的判断（"技术领先"→有事实证据，不是自说自话）
- S 和 O 不重复，W 和 T 不重复

> **输出要求：** S/W/O/T 各 3-5 条，每条不超过 20 字。总数为 12-20 条。

## 报告输出结构

```
一、公司概览（历史沿革、业务结构、股东结构）
二、行业生命周期定位（S 曲线位置、渗透率、关键驱动因子）
三、波特五力分析（行业竞争格局综合评级）
四、护城河分析（宽/窄/无，及依据）
五、财务分析（杜邦 ROE 分解 + 现金流质量 + 警示信号检查）
六、管理层评估（言行一致性、资本配置、治理结构）
七、估值矩阵（2-3 种方法 × 3 情景目标价）
   — 附：可比公司估值对照表
八、ESG 风险评估（可选）
九、SWOT 总成
十、投资结论与核心假设
```

### 内容纪律

1. **所有判断都必须附证据。**"行业前景广阔"→不许出现。"行业未来 3 年 CAGR 约 15% (XX 咨询报告)"→可以。
2. **不回避负面。**必须列出风险清单，每个风险带量化评估（发生概率 × 影响程度）。
3. **估值模型输入参数必须写明假设依据。** 不允许出现"假设 WACC=10%"却不说明原因。
4. **敢给明确结论。** 投资建议三选一：推荐 / 中性 / 回避。
   - "推荐" = 目标价 > 当前价 20%+ 且下行风险可控
   - "中性" = 估值合理/信息不足/等待更佳入场点
   - "回避" = 目标价 < 当前价或存在不可逆的结构性风险
5. **核心假设容错机制。** 每个核心判断标注"什么数据出现说明我错了"，如：
   - "我们看好公司的逻辑是液冷渗透率在 2026 年达到 50%——如果 2026 上半年渗透率未达 30%，这个判断需要修正"
6. **信息不足时诚实标注。** "公司未披露该业务板块的具体毛利率，无法做可比分析。"
7. 禁止出现的词汇：赛道很大、团队优秀、前景广阔、值得期待、任重道远、未来可期。
8. **中文撰写**，专业术语可保留英文。
